# Massilia Connection - Site Vitrine

Ce projet est un site vitrine realise pour un client, autour de la presentation de Massilia Connection et de son reseau de partenaires locaux en region PACA.

## Objectif

Construire une page web claire, moderne et lisible pour :

- presenter l'activite et l'histoire de la structure ;
- valoriser les equipes, le materiel et les references ;
- mettre en avant les partenaires locaux ;
- proposer une section contact simple et directe ;
- fournir une page dediee aux mentions legales.

## Structure du projet

- `massilia_connection_mockup_v2.html` : page principale du site
- `massilia_connection_mockup_v2.css` : feuille de style separee
- `mentions-legales.html` : page dediee aux mentions legales
- `asset/logo/favicon-octopus.png` : favicon (pieuvre)
- `asset/` : images, logos et ressources visuelles

## Points principaux realises

- separation HTML/CSS ;
- amelioration de la structure HTML (`head`, `body`, favicon) ;
- ajout d'animations douces et d'effets de survol pour dynamiser l'interface ;
- optimisation de la lisibilite (contrastes, rythme visuel, hierarchie des textes) ;
- ajout de la section partenaires avec accroche orientee acteurs locaux ;
- personnalisation du footer et des liens de navigation ;
- creation de la page `mentions-legales.html` avec hebergeur OVHcloud ;
- integration d'un favicon pieuvre plus lisible dans l'onglet navigateur.

## Lancer le projet en local

Option simple (double-clic) :

1. Ouvrir `massilia_connection_mockup_v2.html` dans un navigateur.

Option serveur local (recommande) :

1. Se placer dans le dossier du projet.
2. Lancer un serveur HTTP local (exemple : extension Live Server de VS Code).
3. Ouvrir la page principale et verifier les liens internes.
